# 实操案例：行政审批预审

本案例用于 90 分钟专题课的现场演示。目标是在不接入 OA/SAP、不使用真实数据的情况下，用本地 AI Skill 跑完一次行政审批材料预审。

## 文件说明

| 文件 | 说明 |
|---|---|
| `sample-request.md` | 模拟电脑报废回购申请材料 |
| `sample-output.md` | 标准预审输出样例 |
| `demo-script.md` | 讲师 5 分钟演示脚本 |
| `prompt.md` | 可复制运行的提示词 |
| `record-replay-extension.md` | Codex Record & Replay 扩展演示与复盘模板 |

## 教学重点

- AI 做预审，不做最终审批。
- AI 可以检查材料、识别风险、起草话术。
- 资产状态、政策依据、回购价格、系统记录必须人工确认。
- Codex Record & Replay 可以记录一次脱敏演示，帮助团队复盘真实操作路径，并把稳定动作沉淀为 Skill 改进项。

## Record & Replay 扩展

如果课堂时间允许，讲师可以在演示前说明录制范围，只录制本目录的模拟材料、提示词运行和输出对照，不录制真实 OA/SAP/资产/财务/HR 系统。录制结束后，带学员复盘：

- 哪些步骤每次都会重复？
- 哪些判断需要写进 Skill？
- 哪些字段必须脱敏或标注为人工确认？
- 哪些操作可以作为新同事培训的标准路径？
