# 可复制提示词

```text
Use $admin-approval-preflight to review this administrative request.

请输出：
1. 申请摘要
2. 材料完整性检查
3. 风险点
4. 需补充事项
5. 审批意见草稿
6. 给申请人的回复话术
7. 必须人工复核的点

要求：
- 不替代审批人做最终决定；
- 不编造制度、价格、资产状态或系统查询结果；
- 明确区分已提供信息、缺失信息、需要人工确认的信息；
- 涉及 OA/SAP/资产系统的数据，只提示需要查询，不声称已经查询。

申请材料：
[粘贴 sample-request.md 内容]
```
